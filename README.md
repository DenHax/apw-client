# APW Client

Start:

```
git clone https://github.com/DenHax/apw-client
cd apw-client
npm intsall
npm run dev
```

Pages: Employee, Upload, Subsystem, Fuel road

Modal forms: ADD upload, EDIT employee

Components: Modal, aside Navbar, Button, Selector, Table
